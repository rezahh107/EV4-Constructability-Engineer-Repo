# CE Builder Producer Contract Summary

CE emits structured constructability evidence.

For visual-reference packages, CE keeps:

```yaml
connector_layer:
  node: string
  model: string
```

The downstream CE→Builder adapter owns the compact Builder projection:

```yaml
connector_layer: node:model
```

This summary points to `docs/CE_TO_BUILDER_PRODUCER_CONTRACT.md` as the authoritative producer-side document.
